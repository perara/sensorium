# Code of Conduct

## Our standard

This project is committed to a respectful, collaborative environment.

Examples of behavior that help create that environment:

- assuming good intent while staying clear and direct
- sharing technical feedback without contempt
- welcoming questions from people at different experience levels
- documenting context so others can reproduce and learn

Examples of unacceptable behavior:

- harassment, intimidation, or discrimination
- personal attacks or repeated hostile language
- doxxing or publishing private information without permission
- deliberate disruption of discussion or collaboration

## Scope

This code of conduct applies in project spaces, including:

- issue discussions
- pull requests
- code review
- chat or email used for project coordination

## Enforcement

Project maintainers may remove, edit, or reject contributions and comments that
violate this policy.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers
privately rather than escalating in a public thread.
