# sensorium v0.1.1

## Highlights

- improved GitHub Actions reliability for CI, Debian, Alpine, and Arch package
  workflows
- consolidated IMX IPA tuning defaults into a single generic source file
- improved end-to-end media examples for both local files and public sample
  videos

## Included fixes

- public sample MP4 defaults now replace the old private demo URL
- stream helpers accept local files as well as `http(s)` URLs
- raw camera retrieval docs now follow the tested `v4l2-ctl` plus `ffmpeg`
  path
- package workflows are hardened for containerized GitHub runners

## Notes

- this is a patch release focused on packaging, workflow reliability, and repo
  polish
- the public runtime surface and IMX profile catalog remain unchanged from
  `v0.1.0`
