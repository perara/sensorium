# sensorium v0.1.2

## Highlights

- fixes GitHub Release asset publishing for nested package artifacts
- keeps the packaging workflow behavior from `v0.1.1` while ensuring the Alpine
  `.apk` can be attached to releases

## Included fixes

- release upload globs now recurse into downloaded artifact directories
- release metadata now lines up cleanly with the successful package workflow
  outputs

## Notes

- this is a patch release focused on release packaging and asset publishing
- runtime behavior, driver ABI, and IMX profile coverage are unchanged from
  `v0.1.1`
